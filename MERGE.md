# TONGGACHA — сайт

## Что это
Одностраничник на чистом HTML/CSS/JS (без сборки, без зависимостей).
Тема: ночь → рассвет — при скролле вниз чёрный фон плавно сменяется
на светлый, лого автоматически переключается на чёрную версию.

## Структура
```
tonggacha/
├── index.html          ← всё (HTML+CSS+JS в одном файле)
└── assets/
    ├── logo-white.png  ← лого с белым текстом, прозрачный фон (для тёмных секций)
    └── logo-black.png  ← лого с чёрным текстом, прозрачный фон (для светлых секций)
```

## Как запустить
Просто открой `index.html` в браузере — всё работает локально, бэкенда нет.

Для деплоя на GitHub Pages:
1. Залей папку `tonggacha/` в репозиторий (например `operixai/tonggacha`
   или отдельный репо для бренда).
2. В настройках репо: Settings → Pages → Source: `main` / `root`.
3. Сайт появится на `https://<username>.github.io/<repo>/`.

Для своего домена — просто закинь содержимое папки на любой статический
хостинг (Vercel, Netlify, обычный VPS + nginx).

## Что уже работает
- Hero с крупным лого, анимация появления, "лучи" на фоне
- Манифест-блок (текстовый, типографика как акцент)
- Маршрут: Ташкент → Регион → Глобально
- Тизер события с живым обратным отсчётом (сейчас стоит +30 дней от
  текущей даты — placeholder, см. ниже как поменять)
- Галерея-сетка под будущие фото/видео (плейсхолдеры с подписями)
- Форма подписки (пока без бэкенда — просто меняет текст на странице)
- Переход тема "ночь → рассвет" при скролле

## Что нужно будет donastroить
1. **Дата события** — в `index.html` найди:
   ```js
   const target = new Date();
   target.setDate(target.getDate() + 30);
   ```
   Замени на реальную дату:
   ```js
   const target = new Date('2026-08-15T22:00:00');
   ```

2. **Форма подписки** — сейчас она ничего не отправляет, просто меняет
   текст. Подключить реально можно через:
   - тот же Google Sheets Apps Script webhook, что ты делал для Operix
     (структура та же: `fetch(WEBHOOK_URL, {method:'POST', body:...})`)
   - или сервис типа Telegram-бота — на submit формы дёргать твой
     NEXUM/JARVIS бэкенд и писать лиды прямо в Telegram

3. **Галерея** — когда появятся реальные фото/видео с тусовок, ячейки
   `.gallery__cell` в HTML заменяются на `<img>` / `<video>`. Сетка и
   hover-эффекты уже готовы под это.

4. **Шрифт** — сейчас используется системный narrow grotesque
   (Archivo Narrow / Arial Narrow) как замена твоему лого-шрифту. Если
   хочешь точное совпадение с лого — пришли название шрифта или сам
   файл (.woff2/.ttf), подключу через `@font-face`.

## Технические детали
- Чистый CSS (без Tailwind/фреймворков), всё в `<style>` внутри файла
- Анимации через `IntersectionObserver` (scroll-reveal) + CSS keyframes
- Адаптивно вниз до мобилки (галерея/маршрут/меню перестраиваются)
- `prefers-reduced-motion` уважается — анимации отключаются для тех,
  кто это просил в системе
